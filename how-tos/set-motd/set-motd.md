#How-To Set custom motd (message of the day) (cmd):

Ref guide at: [PUTORIUS][1]

1. Check the current motd:
    - `cat /etc/motd`

2. Set the motd:
    - `sudo vim /etc/motd`
    - Type or paste your desired message

3. Nice custom fonts can be found at: [patorjk fonts][2]
